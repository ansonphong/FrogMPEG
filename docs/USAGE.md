# Usage

## GUI

```
frogmpeg-gui.bat
```

- Arrow keys navigate folders/presets/extensions
- `Tab` switches panels
- `S` starts conversion
- `R` refreshes folder list
- `Q` quits

## CLI

```
python -m frogmpeg convert --folder 01-01-01-2025-11-23-A --preset fulldome-2k
python -m frogmpeg list-presets
python -m frogmpeg init
python -m frogmpeg validate
```

The batch file `frogmpeg.bat` forwards all arguments to the CLI.
